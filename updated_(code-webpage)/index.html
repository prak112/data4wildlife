ONE-OFF SETUP
  1. Install Python3, if needed: guides for MacOS and Windows10
  2. From Terminal (MacOS) or Command-line (Windows) install Python dependencies with pip3 install requests
  3. Ask Al Jamieson for the API keys and, using a text editor, add them to the top of the "index.js" and "InstaPostsWithHashtags.py" files in the "scripts" folder
SOURCING HASHTAG POSTS
  1. Using a text editer, add hashtags relevant to IWT at the top of the "InstaPostsWithHashtags.py" file in the "scripts" folder
  2. From Terminal/Command-line, in the "scripts" folder, run the script with python3 InstaPostsWithHashtags.py
  3. Posts found on Instagram with those hashtags will be saved, with their image, caption, user profile, location, etc
  4. Come back to this website to manually review the posts

  1. If using this website tool for the 1st time, or after a while, head here and click 'Request temp access'
  2. Click "Import"
    • Browse to the folder where this code is
    • Open the 'hashtags' folder
    • Open the folder for the specific hashtag you want to manually review
    • Open the file ending with '-page01.json'
  3. The data will load and then be automatically translated into English by Google Translate
  4. Flag each post that you believe to be IWT
  5. Click "Export" to download a CSV file that only includes the flagged posts

  • As a proof of concept this tool grabs just the 1st page of results per hashtag (approx 70). This would be extended to all posts (some had 10k's of posts).
  • This tool to be easily modified to select a differnt target language for automatic translation for analysts around the world.
  • When enough data is collected and manually tagged as accurate IWT, machine learning and AI can be applied to the data to fully automate the collection and alerting of IWT.
